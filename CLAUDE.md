\# PostBot MVP — Israel Post AI Delivery Assistant



\## Project Purpose

University assignment prototype (Reichman University, Entrepreneurship \& Innovation course).

Demonstrates an AI-powered WhatsApp delivery assistant for Israel Post (דואר ישראל).

This is a 7-minute Zoom presentation demo — visual quality and clarity matter most.



\## What PostBot Does

PostBot is an AI assistant that proactively communicates with customers about their parcels.

It runs on WhatsApp/SMS. No app download required.



\## 5 Core Functions

1\. Proactive status alerts — push update every scan event

2\. Anomaly detection — no scan for 48h → auto-opens support ticket

3\. Pre-delivery confirmation + reschedule option

4\. Current events delay warning (war, airport shutdowns, import taxes)

5\. Natural language Q\&A — customer asks anything, PostBot understands



\## Demo Structure (3 Scenarios)

\- Scenario 1: Customer asks for tracking → PostBot responds with status + ETA

\- Scenario 2: PostBot proactively alerts about stuck parcel → customer reschedules

\- Scenario 3: PostBot proactively warns about delay due to reduced flights from China



\## Tech Constraints

\- Single HTML file, no backend, no external dependencies

\- Must run by opening the file in a browser
